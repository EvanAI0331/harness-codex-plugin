# Contributor Covenant Code of Conduct

This project follows the [Contributor Covenant Code of Conduct, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct.html).

In short:

- be respectful
- be constructive
- avoid harassment, discrimination, and abuse
- help keep discussions focused on the code and the product

Project maintainers may remove content or restrict participation when needed to protect the community.
