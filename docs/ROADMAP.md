# Roadmap

## Done

- Single workspace entry point
- Build / run separation
- Task instance layer
- Artifact-first run outputs
- Final deliverable aggregation
- Server-side credential resolution via `credentialRef`
- Demo mode adapter boundaries
- Runtime settings page

## Next

- Harden adapters and their interfaces
- Deepen GitHub capability import and reuse
- Expand final deliverable richness
- Improve run replay and auditability
- Add stronger spec visualization in the workspace
- Broaden smoke coverage around demo flows

## Later

- Production hardening
- Multi-tenant isolation
- Policy-based secret and runtime scoping
- Distributed execution and scaling
- Collaborative review and approval surfaces
- More importable capability packs and skill registries

## Notes

- The project currently ships as an OSS skeleton with a real runtime path.
- Some integrations remain pluggable placeholders by design.
